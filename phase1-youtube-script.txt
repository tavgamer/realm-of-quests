  Title

  I Built a Full RPG Game in ONE Day — Zero Experience (Claude Code)

  Description

  What if I told you I built a full RPG game in a single day with zero game dev experience?

  I used Claude Code (an AI coding assistant) to build "Realm of Quests" — a top-down 2D fantasy RPG that runs right in your browser.

  🎮 PLAY THE GAME: https://tavgamer.github.io/realm-of-quests/
  💻 SOURCE CODE: https://github.com/tavgamer/realm-of-quests

  ⚡ Tech Stack:
  - Phaser 3 (JavaScript game framework)
  - Claude Code (AI coding assistant)
  - WSL + Ubuntu on Windows
  - Plain HTML/JS — no build tools

  🗺️ What's in Phase 1:
  - Walk around Greenwood Village
  - Meet NPCs: Village Elder & Shopkeeper
  - Pixel art character sprites with 4-direction animation
  - HUD with HP, Level, and Gold

  🔮 Coming Next:
  - Phase 2: Combat (sword attacks, enemy AI)
  - Phase 3: Quests & NPC dialog
  - Phase 4: Leveling up & multiple areas
  - Phase 5: Inventory & weapon shop

  Subscribe to follow the full journey! 🚀
