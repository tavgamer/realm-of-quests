# Realm of Quests

A top-down 2D fantasy RPG built with Phaser 3. Explore areas, complete quests, fight enemies, level up, and collect gear!

## How to Run

1. Open a terminal and navigate to the project folder:
   ```bash
   cd ~/learning/realm-of-quests
   ```

2. Start a local web server:
   ```bash
   python3 -m http.server 8000
   ```

3. Open your browser and go to: **http://localhost:8000**

## Controls

- **WASD** or **Arrow Keys** — Move
- **Space** — Attack (coming in Phase 2)
- **E** — Talk to NPCs (coming in Phase 3)
- **I** — Open Inventory (coming in Phase 5)
- **Escape** — Pause Menu (coming in Phase 6)
